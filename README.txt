Fake Lact Unit

Purpose:
    The purpose of this software is to create a simulated lact unit to send data to Lavaro Qlogix device

How it works:
    It will create a modbus server that will connect via ethernet to Qlogix Device.
    
language:
    Written in C++
    
Depenencies:
    g++
    libmodbus5
    libmodus-dev
